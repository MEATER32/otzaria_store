חזרה לחנות

איך מפרסמים תוסף חדש בחנות

מוסיפים תמונה, משלימים את פרטי התוסף בקובץ הנתונים, ובודקים שהקישורים עובדים. כך התוסף יופיע בחנות עם עמוד פרטים מלא.

שלב 1: מעלים תמונות

שמים את קובץ התמונה הראשי, ואם צריך גם צילומי מסך, בתוך assets/images/.

שלב 2: מוסיפים אובייקט ל־JSON

פותחים את data/plugins.json ומוסיפים תוסף חדש לרשימה. אם זה התוסף הראשון, מחליפים את [] ברשימה עם אובייקט אחד. 

[
  {
    "id": "my-plugin",
    "name": "השם של התוסף",
    "shortDescription": "תיאור קצר שיופיע בקוביה.",
    "description": "תיאור מלא שיופיע בעמוד התוסף.",
    "version": "1.0.0",
    "status": "stable",
    "author": "שם המפתח או הצוות",
    "updatedAt": "2026-04-28",
    "compatibleWith": "0.9.88 ומעלה",
    "tags": ["חיפוש", "מראה תואם לאוצריא"],
    "image": "assets/images/my-plugin.svg",
    "screenshots": [],
    "downloadUrl": "https://your-domain.com/plugins/my-plugin.otzplugin",
    "homepage": "https://github.com/USER/REPO",
    "installInstructions": [
      "אם אוצריא מותקנת במחשב, אפשר ללחוץ בחנות על התקנה ישירה.",
      "מורידים את קובץ התוסף.",
      "פותחים את מסך התוספים בתוכנה.",
      "בוחרים התקנה מקובץ.",
      "מאשרים ומפעילים מחדש אם צריך."
    ]
  }
]

שלב 3: מוודאים שקישור ההורדה מתאים גם להתקנה ישירה

כאשר downloadUrl מצביע ישירות לקובץ .otzplugin, החנות יוצרת אוטומטית כפתור "התקנה ישירה לאוצריא" ליד כפתור ההורדה. 

otzaria://plugin/install?url=https%3A%2F%2Fyour-domain.com%2Fplugins%2Fmy-plugin.otzplugin

חשוב להשתמש בקישור ישיר לקובץ התוסף עצמו, ולא לעמוד הורדה כללי.

שלב 4: בודקים 5 דברים

  1. שהתמונה נטענת.
  2. שכפתור ההורדה עובד.
  3. שכפתור ההתקנה הישירה מופיע ועובד, אם הקישור הוא לקובץ .otzplugin.
  4. שעמוד התוסף נפתח.
  5. שאין שגיאת JSON בגלל פסיק מיותר.

כללים קצרים

  1. id צריך להיות באנגלית, בלי רווחים, ועדיף עם מקפים.
  2. status צריך להיות stable, beta או experimental.
  3. downloadUrl צריך להיות קישור ישיר לקובץ .otzplugin אם רוצים התקנה ישירה מתוך אוצריא.
  4. התגית מראה תואם לאוצריא מיועדת לתוספים שמשתמשים בשפת העיצוב של Material 3 ומתאימים חזותית לממשק של אוצריא.
  5. השתמשו בתגיות רק כשהן באמת עוזרות למשתמש להבין מה התוסף נותן.
